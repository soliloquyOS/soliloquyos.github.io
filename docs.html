Soliloquy OS Documentation

Introduction

Welcome to soliloquyOS, a FOSS operating system built from the ground up for VR. This documentation will guide you through the unique concepts and features of soliloquy.

Glossary

Dollhouse
The overall structure of the Soliloquy OS, representing the entire virtual environment.
Diorama
Individual application spaces within the Dollhouse, each representing a different program or function.
Hallways
Transition areas between Dioramas, allowing users to navigate between different applications.
.sol files
The unique executable file format for Soliloquy applications (subject to change).
