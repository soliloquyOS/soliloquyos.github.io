soliloquyOS Devlog


hello world!

Posted on the 2nd of October, 2024 by Marceline

so! this is the start of a pretty exciting project: soliloquyOS. a foss operating system built from the ground up for vr. unlike every other vr os on the market, soliloquyOS is being built from the ground up to be used for 3d interfaces, not floating 2d windows in a 3d space.


i guess i'm starting this project because this is something that i would actually want to use. currently, there's no real reason for me to use vr over a traditional display on a laptop or desktop for my everyday computing tasks. game development, making npr artwork in blender, colour grading in resolve, standard web browsing stuff, etc. but soliloquy would ideally give me some sort of reason to do these tasks in vr.


i've decided to open source this because i believe that the democratisation of software is important. i mean who even benefits from software being completely closed source? besides corporations that want to get stupidly rich, lol. so yeah, feel free to steal this code. it's free under an MIT license. and maybe even contribute if you're feeling kind :)


oh, and just as a small aside here, i don't plan on having backwards compatibility for any other programs. i mean how could you? soliloquy is meant to run applications with 3d interfaces. and i don't know any programs currently, no matter how baisc, that have 3d interfaces. and so, to mark the occasion, i've decided against using MSWIN4.1 as my bdb_oem. instead opting for: 

bdb_oem:                    db 'slqy-os '       ;8 bytes
